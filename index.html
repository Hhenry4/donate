// about me

Hey, I'm Hhenry — I build games and web projects in my spare time because I genuinely love making things. What started as tinkering with HTML turned into full mystery game experiences with custom mechanics, background music, encrypted mystery data, and replayable story lines.

Every project I ship is free to play. I don't put games behind paywalls. But building takes time, and tools cost money. Donations let me spend more hours on the thing I love most: making games people enjoy.

Currently a solo developer handling everything — game logic, UI design, server code, sound integration. If you've played one of my games and want to see more, this is how you make that happen.
